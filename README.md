# ShelfSpace

## Проблема
Любителям книг складно зберігати власні враження від прочитаних книг, оцінки та рецензії в одному місці. Через це відгуки губляться, а користувачам важко відстежувати прочитані книги та ділитися своєю думкою.

## Рішення
Веб-застосунок на Java для створення, перегляду, редагування та видалення рецензій на книги. Користувач може додавати книги, ставити оцінки, писати відгуки та переглядати список прочитаного.

## Цільовий користувач
Користувачі, які читають книги, ведуть список прочитаного та хочуть зберігати власні рецензії й оцінки.
